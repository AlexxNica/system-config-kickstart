Chapter 13. Saving the File

After you have finished choosing your kickstart options, click the Save File button. A dialog box similar to Figure 13-1 will appear to allow you to review your choices before saving the file.

Figure 13-1. Kickstart Configurator - Confirm Options

If you are happy with your choices, click the Save File button within the dialog box. A save file dialog box will appear and allow you to choose where to save the file. The default file name to save it as is ks.cfg.

After saving the file, create a boot disk from boot.img for CD-ROM installations or bootnet.img for network installations. Copy the kickstart file (ks.cfg) on the boot disk. Boot from the disk and type linux ks=floppy to start the kickstart installation. To learn about different methods for starting a kickstart installation refer to the Official Red Hat Linux Customization Guide.
