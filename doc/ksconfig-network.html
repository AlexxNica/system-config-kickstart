Chapter 6. Network Configuration

Figure 6-1. Network Configuration

There are three network configuration options: DHCP, Static IP, and None. If there is not an Ethernet card in the system, choose None.

Networking is only required if you choose a networking-type installation method (NFS or FTP). If you are unsure which to choose, choose None. Networking can always be configured after installation with the Red Hat Network Configurator using the command (redhat-config-network.

If you select Static IP, you must provide additional networking information in the table below the network types.
