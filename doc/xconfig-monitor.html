Monitor

After configuring the video card, click on the Monitor tab shown in Figure 9-3 and select the monitor for the system. You can specify the horizontal and vertical sync rates instead of specifying a monitor by checking the Specify hysnc and vsync instead of monitor option. This option is useful if the monitor for the system is not listed. Notice that when this option is enabled, the monitor list is disabled.
